/api/v1/rundeck/job¶

POST /api/v1/rundeck/job/(job-name)/(instance)¶

Executes the Rundeck job specified by (job-name) on the target (instance).

The followin example covers the execution of a job named get-snapshots which returns the list of snapshots from a certain instance which could be used to launch a restore.

Example request:

curl -i -u $AUTH -X POST https://<server>:5443/api/v1/rundeck/job/get-snapshots/pinocho
