/api/v1/host¶

GET /api/v1/host/names/<name>¶
Returns a json with the memory assigned to each host.

Example request:

curl -i -X GET https://<server>:<port>/api/v1/host/names/<name>

Example response:

HTTP/1.1 200 OK
Date: Thu, 01 Dec 2016 17:22:06 GMT
Content-Length: 30
Etag: "ffd3be949a2dd4b26086119241118ae42ef05432"
Content-Type: application/json; charset=UTF-8
Server: TornadoServer/4.2

{
        "response": [
                {
                        "memory": 42
                }
        ]
}
Query Parameters:
 
  • name – Host name
Response Headers:
 
  • Content-Type – application/json; charset=UTF-8
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Name (host) not found in system
  • 500 Internal Server Error – Backend error (api or postgrest or database)
POST /api/v1/instance/host/names/<name>¶

Example request:

curl -k -i -X POST -u <usernmae>:<password> https://<domain>:<port>/api/v1/host/names/<new_name>?memory=<memory_in_MB(int)> or curl -k -i -X POST -u <usernmae>:<password> -H 'Content-Type: application/x-www-form-urlencoded' https://<domain>:<port>/api/v1/host/names/<new_name> -d 'memory=<memory_in_MB(int)>'

Example response:

HTTP/1.1 201 Created
Date: Thu, 01 Dec 2016 17:35:07 GMT
Content-Length: 0
Content-Type: text/html; charset=UTF-8
Server: TornadoServer/4.2
Query Parameters:
 
  • name – Host name
Request Headers:
 
  • Accept – application/json
Response Headers:
 
  • Content-Type – application/json
  • Charset – UTF-8
Status Codes:
  • 201 Created – Host mapping successfuly created
  • 400 Bad Request – Error when the format or the argument is not correct
  • 409 Conflict – Error when inserting duplicate names
